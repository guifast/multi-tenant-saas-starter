# Demo Walkthrough

Target duration: 60 to 120 seconds.

1. Start the project locally with Docker, migrations and seed data.
2. Open `http://localhost:3000`.
3. Log in with `owner@example.com`.
4. Select the `acme` workspace.
5. Create a project.
6. Open the audit log and show the project event.
7. Switch to the `beta` workspace.
8. Show that `acme` projects are not visible.
9. Log out.
10. Log in with `viewer@example.com`.
11. Attempt a mutation and show the safe forbidden response.

Do not commit the video file. Add an external video link to the README only after recording it.
